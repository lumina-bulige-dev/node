# Admin Control Panel

Independent workspace for the administrative control panel UI.

## Structure

- `src/app/(control-panel)/page.tsx`: control panel entry page with 2-pane resizable layout.
- `src/components/control-panel/*`: UI placeholders for target selection, change modules, preview, guardrails, and approval.
- `src/domain/decision/types.ts`: centralized domain types for decision drafts.
